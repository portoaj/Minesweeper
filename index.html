1. Why I Made This Game

I had a free period and nothing else to do. Minesweeper is pretty fun. I like Minesweeper.

2. What I Learned

I should make it show up when the game starts. The input didn't work/ needed to be fixed. Android Studio is cool.

3. Architecture of My Game

There is a 2D array of Tile class instances which communicate with the Input Handler and Main class to update everything. The Main class updates the visual in Draw. If the game ends the input handler is ignored for 5 seconds before the game restarts.
